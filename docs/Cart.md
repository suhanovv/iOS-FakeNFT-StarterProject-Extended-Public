# Shopping Cart

## Milestone 1/3 (Estimate Time: 19h / Fact Time:)

- Проанализировать дизайн корзины (все экраны, состояния, модальные окна, переходы) (Estimate Time: 1h / Fact Time:)
- Перенести ассеты в проект (иконки платежных систем, изображения, NFT плейсхолдеры) (Estimate Time: 1h / Fact Time:)
- Проанализировать API корзины и оплаты (методы orders, currencies, payment) (Estimate Time: 6h / Fact Time:)
- Описать интерфейсы (протоколы) и DTO для сервисов: (Estimate Time: 6h / Fact Time:)
    - управления заказом (получение/обновление order)
    - получения валют (currencies list)
    - обработки платежа (set currency, pay)
- Подготовить моки (Estimate Time: 1h / Fact Time:)
- Сверстать компонент "Карточка покупки" на моках для экрана "Корзина" (предпросмотр, название, цена, рейтинг, кнопка удаления) (Estimate Time: 2h / Fact Time:)
- Сверстать компонент "Карточка валюты" на моках для экрана "Выбор оплаты" (предпросмотр, название, код, состояния выбрана/не выбрана) (Estimate Time: 1h / Fact Time:)
- Сверстать компонент "Окно подтверждения удаления" на моках (превью товара, вопрос, две кнопки) (Estimate Time: 1h / Fact Time:)

## Milestone 2/3 (Estimate Time: 26h / Fact Time:)

- Сверстать экран "Корзина" на моках (список карточек, блок перехода к оплате, количество и сумма) (Estimate Time: 4h / Fact Time:)
- Сверстать состояние "Корзина пуста" на моках (Estimate Time: 1h / Fact Time:)
- Сверстать экран "Выбор способа оплаты" на моках (список валют, блок согласия с условиями и кнопка оплатить) (Estimate Time: 4h / Fact Time:)
- Сверстать экран "Успешная оплата" на моках (котик, текст, кнопка возврата) (Estimate Time: 1h / Fact Time:)
- Реализовать навигацию между экранами (Корзина → Выбор оплаты → Успех) (Estimate Time: 4h / Fact Time:)
- Реализовать логику сортировки товаров через ActionSheet (по цене, рейтингу, названию) (Estimate Time: 4h / Fact Time:)
- Реализовать логику выбора валюты (выделение карточки, смена состояния) (Estimate Time: 2h / Fact Time:)
- Реализовать отображение модального окна подтверждения удаления (Estimate Time: 2h / Fact Time:)
- Реализовать отображение Alert при ошибке оплаты (Estimate Time: 2h / Fact Time:)
- Реализовать обновление счетчиков и суммы при изменении корзины (Estimate Time: 2h / Fact Time:)

## Milestone 3/3 (Estimate Time: 31h / Fact Time:)

- Спроектировать общую архитектуру сервисов (вынесение общей логики, протоколы) (Estimate Time: 8h / Fact Time:)
- Реализовать сервис управления заказом (получение order, добавление NFT в order) (Estimate Time: 3h / Fact Time:)
- Реализовать сервис получения валют (получение списка currencies, кеширование) (Estimate Time: 3h / Fact Time:)
- Реализовать сервис обработки платежа (set currency, выполнение pay) (Estimate Time: 3h / Fact Time:)
- Подвязать сервисы к экранам с учетом пагинации, убрать моки (Estimate Time: 8h / Fact Time:)
- Реализовать обработку ошибок (NFT уже продан, недостаточно средств, timeout, network errors) (Estimate Time: 6h / Fact Time:)
